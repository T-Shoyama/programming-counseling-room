# プログラミング相談室

# 概要

プログラミングの悩みを投稿できるサイトを作りました。

# 使い方

①	プログラミングの悩みを投稿できる
②	投稿された悩みに対してコメントを送ることができる
③	悩みを投稿したら、マイページで表示
④	気になった投稿はチェックして、マイページで一覧表示させる
⑤	ユーザーのお気に入り登録機能
⑥	ユーザーが投稿した内容や、チェックした投稿が見られる。
